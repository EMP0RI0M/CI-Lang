CI-Lang is a next-generation programming language designed to handle computational chaos and entropy as core primitives.

## Features
- **Entropy-Aware Syntax**: Native support for probabilistic logic and adaptive execution.
- **FluxVM**: A specialized execution engine with an integrated entropy register.
- **Modern Grammar**: Formal BNF specification optimized for subword tokenization.

## Project Structure
- `docs/`: Technical whitepaper and FluxVM architecture.
- `grammar/`: Formal BNF specification.
- `src/`: FluxVM prototype implementation.
- `examples/`: Sample CI-Lang programs.

## Getting Started
1. Run the FluxVM prototype: `python src/fluxvm.py`
2. Run the benchmarking suite: `python src/benchmark.py`

## Vision
To replace Python as the dominant language for chaos-aware systems and adaptive computing.
